Installing

~~~~~~~~~~~~~~~~~~~~~~~~
.. code:: bash

    git clone `https://github.com/DSilva27/cryo_em_SBI.git`

Navigate to the cloned repository and install the module
~~~~~~~~~~~~~~~~~~
.. code:: bash
